Кастомизация

С помощью переменных в CSS, клиенту будет куда проще подстроить сайт под свою цветовую гамму.

/* Font colors */
--font-color: #000000;
--font-invert-color: #dfdfdf;
--font-secondary-color: #8d8d8d;
--font-color-hover: #a9d0ff;

/* Background colors */
--background-color: #ffffff;
--background-invert-color: #252525;
--background-secondary-color: #90c3e0;

/* Effects colors */
--shadows-color: #00000073;
--box-border-radius: 3px;
--animation-speed: 0.25s;

И тому подобное, специально для клиентов была написана документация, чтобы ответить на вопросы по редактированию функционала.
